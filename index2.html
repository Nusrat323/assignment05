Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM.
1. getElementById(): Selects an element by its id.
2. getElementsByClassName(): Selects elements by class name.
3. getElementsByTagName(): Selects elements by tag name.
4. querySelector(): Selects the first matching element.
5. querySelectorAll(): Selects all matching elements.
6. childNodes: Selects all child nodes of an element.
7. parentNode: Selects the parent element of a given element.


Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Gets or sets HTML content (including tags).
innerText: Gets or sets visible text, considering styles.
textContent: Gets or sets all text, ignoring styles and tags.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the target element and "bubbles up" to its parent elements, one by one, until it reaches the root. This means that if you click on an element, the event will also be triggered on its parent, grandparent, and so on, unless it's stopped.

Question-5: How do you create, add, and remove elements using JavaScript?

To create, add, and remove elements using JavaScript, first use document.createElement() to create a new element. For example, const newElement = document.createElement("div") creates a new div element. To add this element to the page, you use appendChild() or insertBefore() on the parent element, like parentElement.appendChild(newElement) to append it to a parent. To remove an element, you use removeChild() on its parent, such as parentElement.removeChild(elementToRemove) to remove an element from the DOM.
